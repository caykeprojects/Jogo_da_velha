# Jogo da Velha Web

Projeto simples de `Jogo da Velha` com HTML, CSS e JavaScript.

## Estrutura do projeto

- `index.html` - página principal com o tabuleiro e a área de exibição do jogador atual.
- `index.js` - lógica do jogo, controle de turno, verificação de vitória e empate.
- `stye.css` - estilos do tabuleiro, botões e layout da página.

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Clique em uma das nove casas para marcar `X` ou `O`.
3. O jogo alterna entre os dois jogadores automaticamente.
4. Quando um jogador vencer ou o jogo empatar, uma mensagem aparece e o tabuleiro é reiniciado.

## Regras do jogo

- Dois jogadores (`X` e `O`) jogam alternadamente.
- O primeiro jogador a completar três marcas em linha vence.
- Se todas as casas estiverem preenchidas sem vencedor, o jogo empata.

## Observações

- O arquivo CSS atual está nomeado como `stye.css`.
- O jogo é executado diretamente no navegador, sem servidor ou instalação necessária.
